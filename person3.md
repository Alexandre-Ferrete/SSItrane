## 2. crypto/ecdh.py

**Purpose:** Implement Elliptic Curve Diffie-Hellman for Perfect Forward Secrecy (PFS).

### Class: `ECDHKeyExchange`

```python
class ECDHKeyExchange:
    def __init__(self, curve: str = "secp384r1")
        # Initialize with elliptic curve (secp256r1, secp384r1, or secp521r1)

    def generate_keypair(self) -> Tuple[bytes, bytes]
        # Generate ephemeral ECDH keypair
        # Returns: (private_key_pem, public_key_pem)

    def derive_shared_secret(self, private_key_pem: bytes, peer_public_key_pem: bytes) -> bytes
        # Perform ECDH key agreement
        # Returns: 48-byte shared secret (using HKDF to derive bytes)

    def generate_ephemeral_keypair(self) -> Tuple[bytes, bytes, object]
        # Generate keypair and return with curve object for later use
        # Returns: (private_key, public_key, curve)
```

### Requirements
- Use `cryptography.hazmat.primitives.asymmetric.ec`
- Support curve `SECP384R1` (recommended) or `SECP256R1`
- Derive exactly 32 bytes from shared secret using HKDF
- Return keys in PEM format for serialization

## 3. crypto/symmetric.py

**Purpose:** AES-GCM authenticated encryption for message confidentiality and integrity.

### Class: `SymmetricEncryption`

```python
class SymmetricEncryption:
    def __init__(self, key: bytes)
        # Initialize with 32-byte AES-256 key

    @staticmethod
    def generate_key() -> bytes
        # Generate random 32-byte AES-256 key

    def encrypt(self, plaintext: bytes, associated_data: bytes = None) -> Tuple[bytes, bytes, bytes]
        # Encrypt data using AES-256-GCM
        # Returns: (ciphertext, nonce, tag)

    def decrypt(self, ciphertext: bytes, nonce: bytes, tag: bytes, associated_data: bytes = None) -> bytes
        # Decrypt AES-256-GCM data
        # Returns: plaintext
        # Raises: InvalidTag if authentication fails
```

### Requirements
- Use `cryptography.hazmat.primitives.ciphers.aead.AESGCM`
- Nonce: 12 bytes (cryptographically random)
- Tag: 16 bytes (automatically generated by AESGCM)
- Support Associated Authenticated Data (AAD) for message binding

```python
class HKDF:
```

## 5. crypto/hybrid.py

**Purpose:** Combine ECDH key exchange with AES-GCM encryption for secure messaging.

### Class: `HybridEncryption`

```python
class HybridEncryption:
    def __init__(self, key: bytes = None)
        # Initialize with session key, or generate new one

    @staticmethod
    def generate_ephemeral_keypair() -> Tuple[object, object]
        # Generate ephemeral keypair for ECDH

    @staticmethod
    def derive_session_key(private_key: object, peer_public_key: object, context: bytes = b"") -> bytes
        # Derive session key from ECDH exchange
        # Returns: 32-byte session key

    def encrypt_message(self, plaintext: bytes, associated_data: bytes = None) -> Dict[str, bytes]
        # Encrypt message with session key
        # Returns: {
        #     "ciphertext": ...,
        #     "nonce": ...,
        #     "tag": ...
        # }

    def decrypt_message(self, encrypted: Dict[str, bytes], associated_data: bytes = None) -> bytes
        # Decrypt message
        # Returns: plaintext
```

### Encryption Flow
1. Generate ephemeral ECDH keypair
2. Exchange public keys with peer
3. Derive shared session key via ECDH
4. Use HKDF to derive 32-byte AES key
5. Encrypt data with AES-256-GCM

### Requirements
- Use `cryptography.hazmat.primitives.asymmetric.rsa`
- Use OAEP padding (not PKCS1v15)
- Hash algorithm: SHA-256 for OAEP

**Purpose:** ECDSA digital signatures for message authenticity and integrity.

### Class: `ECDSASigner`

```python
class ECDDSASigner:
    def __init__(self, private_key_pem: bytes = None, curve: str = "secp384r1")
        # Initialize signer with private key or generate new

    def generate_keypair(self) -> Tuple[bytes, bytes]
        # Generate ECDSA keypair
        # Returns: (private_key_pem, public_key_pem)

    def sign(self, message: bytes) -> bytes
        # Create ECDSA signature (deterministic)
        # Returns: 96-byte signature (for secp384r1)

    def verify(self, message: bytes, signature: bytes, public_key_pem: bytes) -> bool
        # Verify ECDSA signature
        # Returns: True if valid, False otherwise
```

### Requirements
- Use `cryptography.hazmat.primitives.asymmetric.ec`
- Use `ec.ECDSA` with `hashes.SHA384()` or `hashes.SHA256()`
- Implement deterministic signatures when possible

```python
    def sign_csr(self, public_key_pem: bytes, subject: Dict[str, str]) -> bytes
        # Sign certificate signing request
        # Returns: signed certificate pem

    def get_ca_certificate(self) -> bytes

    def get_ca_public_key(self) -> bytes
        # Return CA public key PEM
```

### Class: `Certificate`

```python
class Certificate:
    def __init__(self, cert_pem: bytes)
        # Load certificate from PEM

    @staticmethod
    def from_pem(pem_data: bytes) -> 'Certificate'
        # Create certificate from PEM data

    def verify(self, ca_cert_pem: bytes) -> bool
        # Verify certificate is signed by CA

    def get_subject(self) -> Dict[str, str]
        # Get certificate subject fields

    def get_public_key(self) -> bytes
        # Extract public key from certificate

    def is_valid(self) -> bool
        # Check if certificate is not expired

    def get_serial_number(self) -> int
        # Get certificate serial number

    def to_pem(self) -> bytes
        # Return PEM-encoded certificate
```

### Certificate Requirements
- Use `cryptography.x509` module
- Self-signed CA certificate for internal PKI
- User certificates signed by CA
- Include: Common Name (CN), Organization, Serial Number
- Validity period: 3650 days (CA), 365 days (users)

## 16. Security Considerations

1. **Never hardcode keys or secrets**
2. **Use secure random generation for all nonces/IVs**
3. **Validate all inputs to cryptographic functions**
4. **Handle `InvalidTag` exceptions properly**
5. **Use constant-time comparison for signature verification**
6. **Clear sensitive data from memory when done** (if applicable)
