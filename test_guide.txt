===============================================
TESTING GUIDE - E2EE Chat System
===============================================

== PREREQUISITES ==
1. Terminal 1: Start server
   cd ProjectSSI
   python -m src.server

2. Terminal 2: Start Client 1 (Alice)
   python -m src.client
   Host: localhost
   Port: 5555

3. Terminal 3: Start Client 2 (Bob)
   python -m src.client
   Host: localhost
   Port: 5555

===============================================

== TEST 1: Registration ==
Commands for Alice:
register alice password123
> Expected: [OK] Registration successful!

Commands for Bob:
register bob secure456
> Expected: [OK] Registration successful!

Invalid registrations:
register ab password123
> Expected: [ERROR] Invalid username format

register testuser password
> Expected: [ERROR] Password must be at least 8 chars with letters and digits

register alice abc12345
> Expected: [ERROR] Username already exists

===============================================

== TEST 2: Login ==
Commands for Alice:
login alice password123
> Expected: [OK] Login successful!

Commands for Bob (new terminal):
login bob secure456
> Expected: [OK] Login successful!

Failed login:
login alice wrongpassword
> Expected: [ERROR] Invalid credentials

login nonexistent pass1234
> Expected: [ERROR] Invalid credentials

===============================================

== TEST 3: Private Messaging ==
Alice terminal:
users
> Expected: Shows online users (alice, bob)

msg bob Hello Bob! This is encrypted.
> Expected: [STATUS] Message sent

Bob terminal:
> Should see: [New message from alice]: Hello Bob! This is encrypted.

Bob reply:
msg alice Hi Alice! Message received.
> Expected: [STATUS] Message sent

Alice terminal:
> Should see: [New message from bob]: Hi Alice! Message received.

===============================================

== TEST 4: Offline Messages ==
Bob logs out:
logout
> Expected: [OK] Logged out successfully

Alice (while Bob offline):
msg bob Hello Bob, are you there?
> Expected: [STATUS] Message stored for offline delivery

Bob logs back in:
login bob secure456
> Should see: You have 1 offline message
> Should see: [New message from alice]: Hello Bob, are you there?

===============================================

== TEST 5: Group Chat (Rooms) ==
Alice:
create_room security
> Expected: [OK] Room 'security' created

rooms
> Expected: Shows 'security' room

join security
> Expected: [OK] Joined room 'security'
> Shows members list

Bob:
rooms
> Expected: Shows 'security' room

join security
> Expected: [OK] Joined room 'security'

Alice:
> Should see: [NOTIFICATION] bob joined security

Now in room, type message:
Hello everyone in security room!
> Expected: Message sent to room

> Should see: [security] alice: Hello everyone in security room!

Bob in room:
Hello Alice!
> Expected: Message sent to room

Alice:
> Should see: [security] bob: Hello Alice!

> Expected: [OK] Left room 'security'

Bob:
> Should see: [NOTIFICATION] alice left security

== TEST 7: Server Admin Commands ==
In server terminal (Terminal 1):

> Shows server stats

users
> Shows all registered users

> Shows currently online users

rooms
> Shows active rooms

ban alice
> User alice banned

unban alice
> User alice unbanned

===============================================

== TEST 8: Security Features ==
1. PFS (Perfect Forward Secrecy):
   - Each session uses ephemeral ECDH keys
   - Old messages cannot be decrypted if session key is compromised

2. E2EE:
   - Server never sees plaintext messages
   - Messages encrypted client-side with AES-256-GCM

3. Certificate Verification:
   - Each user gets X.509 certificate from server CA
   - Certificates verify user identity

===============================================

== TEST 9: Error Handling ==
Try to msg nonexistent user:
msg invaliduser Hello
> Expected: [ERROR] Recipient not found

Try to join non-existent room:
join nonexistent
> Expected: [ERROR] Room does not exist

Try to create existing room:
create_room security
> Expected: [ERROR] Room already exists

Try to send to yourself:
msg alice Hello myself
> Expected: [ERROR] Cannot send to yourself
